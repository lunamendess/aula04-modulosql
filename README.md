# Mod-04-Analise-Dados

Projeto Resília - Programadores Cariocas, Referente ao módulo 04.

A proposta do projeto foi modelar um banco de dados sobre um assunto pré-definido, o escolhido pelo nosso Squad foi : Game of Thrones,
onde pudemos analisar seguintes questões : 

1- Frequência de diretores/Episódios;
2- Média de aprovação  por temporada;
3- Densidade de casas por região;
4- Média minutos/temporada;
5- 5 Episódios mais extensos da série;

Utilizamos o programa Xampp para termos acesso ao Phpmyadmin e conseguirmos utilizar o método "SET" para as pesquisas.

Todas os métodos de consultas utilizados estão dispostos nos Slides de apoio para o projeto.
